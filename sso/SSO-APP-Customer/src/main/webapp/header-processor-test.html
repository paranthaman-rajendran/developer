🔐 HTTP Header Processor - Backend Cookie Setter

About This Tool

This tool demonstrates the HeaderProcessorServlet which:

  • Extracts HTTP request headers (User-Agent, Accept, etc.)
  • Sets them as backend-only (HttpOnly) cookies
  • Prevents JavaScript access for security (XSS protection)
  • Returns a JSON response showing processed headers and set cookies

Endpoint: GET/POST /api/process-headers

What to Look For:

  • processedHeaders: Headers extracted from your request
  • cookiesSet: Headers now stored as backend-only cookies
  • HttpOnly flag: Prevents JavaScript from accessing these cookies
  • Check your browser's Developer Tools (F12) → Application → Cookies to see them
