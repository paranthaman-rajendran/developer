# Link Aggregator Application

A simple Spring Boot application with Thymeleaf that serves as a central portal to access multiple sub-applications.

## Features

- Central dashboard for accessing multiple applications
- Responsive design that works on desktop and mobile
- Modern gradient UI with smooth animations
- Easy to configure and customize

## Technologies

- Java 8
- Spring Boot 2.1.18
- Thymeleaf
- Gradle 6.9.4

## Prerequisites

- JDK 8 or higher
- Gradle 6.9.4 (or use the included Gradle wrapper)
